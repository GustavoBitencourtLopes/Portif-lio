Sobre o Projeto

Este projeto simula uma sessão de recarga de veículos elétricos, permitindo inserir tempo de recarga, potência do carregador, tipo de usuário e horário. Com base nesses dados, o sistema calcula automaticamente a energia consumida e o valor total da sessão — desenvolvido para o desafio GoodWe & FIAP.

Funcionalidades

  • Cálculo de energia consumida (kWh)
  • Tarifação por horário (normal ou pico)
  • Desconto para usuários premium
  • Validação dos dados inseridos
  • Resumo completo da sessão

Tecnologias e Ferramentas

JavaScript — lógica de cálculo e regras de tarifação HTML — estrutura da interface CSS — estilização da aplicação
