## 📌 Sobre o projeto

Este projeto foi criado para treinar na prática o que venho aprendendo na FIAP, fora da sala de aula — desde a estruturação de um layout do zero até a integração com um banco de dados real para deixar o formulário de contato funcional.

O site é dividido nas seções:

- **Início** — apresentação rápida
- **Sobre Mim** — formação, habilidades e experiência atual
- **Experiência Profissional** — atuação como Jovem Aprendiz na Aramis
- **Formação** — FIAP e SENAC
- **Habilidades** — tecnologias que domino
- **Projetos** — projetos acadêmicos desenvolvidos até aqui
- **Contato** — formulário funcional integrado a um banco de dados

---

## 🛠️ Tecnologias utilizadas

- **HTML5** — estrutura semântica do site
- **CSS3** — estilização, responsividade e animações (sem frameworks, CSS puro)
- **JavaScript** — interatividade, modais de projetos, validação de formulário e integração com a API
- **Supabase** — banco de dados utilizado para armazenar as mensagens enviadas pelo formulário de contato
- **Font Awesome** — ícones utilizados ao longo do site
- **Git e GitHub** — versionamento e hospedagem do código
- **Vercel** — deploy e hospedagem do site

---

## ⚙️ Funcionalidades

- Layout responsivo, adaptado para desktop e mobile
- Animações sutis (elementos flutuantes, transições e efeitos de hover)
- Modais com detalhes de cada projeto
- Botão de voltar ao topo
- Formulário de contato funcional, que envia os dados diretamente para um banco de dados (Supabase) via API REST
